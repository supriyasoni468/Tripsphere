# 🌍 TRIPSPHERE – Discover India

TRIPSPHERE is a full-stack travel and tourism web application designed to help users explore destinations across India, discover travel packages, access essential travel services, and get emergency assistance while traveling.

## 🚀 Features

*  Explore popular tourist destinations in India
*  Browse travel packages and offers
*  Travel information and destination details
*  Emergency assistance and support system
*  Responsive and user-friendly interface
*  Interactive navigation and search functionality
*  Multimedia content integration
*  Full-stack architecture using Node.js and JavaScript

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/supriyasoni468/Tripsphere.git
cd Tripsphere
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Configure Environment Variables

Create a `.env` file inside the `backend` folder:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

### Run Backend Server

```bash
npm start
```

or

```bash
node index.js
```

### Run Frontend

Open:

```text
frontend/index.html
```

or use a local development server such as VS Code Live Server.
